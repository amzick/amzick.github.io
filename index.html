Intro

After receiving a BA in Electronic Music and Multimedia from Brown University, completing Vancouver Film School's Sound Design for Visual Media program and working in New York's film industry for 7 years, I felt it was time for a new challenge. I completed App Academy's rigorous and intensely competetive Coding Boot Camp in February, 2019.

I love working as a software engineer. Programming requires a unique combination of quantitative reasoning and creativity; striving to balance the two approaches to problem solving is energizing and continually propels me to develop my skills and knowledge.

Portfolio

Let's Go Eros: Live | Github

Let's Go Eros is a single page web app clone of OKCupid built in Rails 5, SQL and React / Redux. I streamlined the user create process by constructing custom routes utilizing regex to allow for real-time validation of user inputs. I integrated the Google Maps API to prioritize matches near the user, personalizing the user's experience and reducing rendering times by not fetching irrelevant data. To demonstrate performance, searchability, and scalability of the app, I built extensive seed data using the Faker and RubyStats gems. Finally, I wrote an algorithm that calculates users' match percentages with other users based on responses to questions in different categories.

Ohio Jackson - The Game: Live | Github

Ohio Jackson (no affiliation with Indiana Jones) is a simple, nostolgic browser video game built in JavaScript and rendered with the HTML5 Canvas API. I built player movement with event listeners and used requestAnimationFrame to render and animate colorful sprites. I leveraged Object Oriented Programming principles to write DRY, modular code, in order to flexibly and easily populate the game with projectiles and items in response to the player's score. By using sets to keep track of game objects I ensure efficient memory usage by quickly removing out of bounds items, allowing for smooth and fast game performance.

About

Resume

I first became entralled with programming on accident while attending Brown University. Part of the requirements of the Electronic Music and Multimedia curriculum were to take one course in the computer sciences department. The logic of Matlab and C++ allowed for a welcome exercise of different parts of the brain from my music course load.

There was still plenty of room for programming over in the music department though, and I took several semesters of instruction in Max / MSP, dabbled in Processing and Arduino, and continued learning audio-oriented programming languages while completing Vancouver Film School's Sound Design for Visual Media program in 2012. As film work became more and more unsatisfying, my thoughts would often turn to programming and software engineering as an new career path, and after conferring with several friends who are in the industry or have completed coding bootcamps, App Academy was a no-brainer.
